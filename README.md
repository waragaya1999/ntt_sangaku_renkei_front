# コミットメッセージ

-   コミットメッセージは以下のフォーマットで記述する
    -   [add] 新規（ファイル）機能追加
    -   [fix] バグ修正
    -   [merge] マージ
    -   [move] ファイル移動
    -   [remove] 削除（ファイル）
    -   [rename] ファイル名変更
    -   [update] 機能修正（バグではない）
    -   [upgrade] バージョンアップ
